## 📝 About This Repository

This repository contains the practical code developed during the course. Its main goal is to consolidate the basic concepts of the C# language and application development.


## 💻 How to Run the Code

Since this project was built using the **.NET CLI**, you can easily run it from your terminal.

1. Ensure you have the [.NET SDK](https://dotnet.microsoft.com/download) installed.
2. Clone this repository and navigate to the project folder.
3. Run the following command in the terminal:

```bash
dotnet run
```
